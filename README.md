# Catálogo de Filmes 

Projeto feito com React + Vite + JSON Server.

## Funcionalidades
- Cadastro de filmes
- Lista de filmes
- Imagem via URL
- Sinopse e avaliação

## Tecnologias
- React
- Vite
- JSON Server
- CSS

## Sobre o projeto
Aplicação de catálogo de filmes onde é possível cadastrar e visualizar filmes com imagem, categoria e nota.
